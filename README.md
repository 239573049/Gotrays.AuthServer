## 使用说明

1.  修改Gotrays.DbMigrator项目配置文件的中的连接字符串
2.  启动Gotrays.DbMigrator项目将同步数据库结构
3.  数据库迁移完成以后运行项目 启动 Gotrays.AuthServer项目
